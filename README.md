# ML_Iris_Flowers
This project is mainly about using some ML knowledge to analysis and explore the Iris Flowers data.  
The main code is in "TASKS/TASKs-LAB1.ipynb"


主要对鸢尾花数据进行探索，并将数据可视化（用散点图和直方图等）。
对数据进行PCA主成分分析法以做降维处理。
利用决策树方法预测新数据。
